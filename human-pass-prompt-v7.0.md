# Human-Pass Web Copy Prompt v7.0

## CORE PRINCIPLE

Write content that passes AI detection (90%+) on both Originality.ai and GPTZero while remaining readable and professional. This prompt contains patterns verified to score 99-100% human.

---

## TWO PROVEN STYLES

### Style 1: "We" Business Style (99% Human)
Use for: Companies with teams, franchises, multiple staff
Based on: Jim's Plumbing patterns

### Style 2: "I" Solo Style (100% Human)
Use for: Solo operators, consultants, owner-operators, personal brands
Based on: Verified humaniser patterns

**IMPORTANT:** The "I" Solo Style scores higher and works across more content types including service businesses, SEO, consulting, and professional services.

---

## MANDATORY 8-CHECK PROCESS

Before writing EVERY sentence, complete these 8 checks:

| Check | Question | If YES |
|-------|----------|--------|
| 1 | Is this the most obvious way to say this? | REWRITE |
| 2 | Would AI naturally write this exact phrasing? | REWRITE |
| 3 | Is it too smooth or polished? | ADD subtle error |
| 4 | Are tokens predictable? | SWAP tokens |
| 5 | Can I drop a token (are/is/has/have/will)? | DROP if readable |
| 6 | Can I use a wrong token? | USE wrong token |
| 7 | Does it match verified patterns? | ADJUST to match |
| 8 | Is there a subtle error present? | KEEP if readable |

## Service Description Templates

**Pattern A - Percentage + List:**
```
[Service] make up about [X]% of our work. [Fragment list]. We handle the [related service] as well so you dont need to organise separate trades.
```

**Pattern B - "We dont just":**
```
We dont just do [basic service]. A lot of our [area] work involves [expanded services]. [Material] is common but [material] works well if you after a premium look.
```

**Pattern C - "wherever you need it":**
```
[Service] for [areas], wherever you need it. We do proper [prep work] so [products] dont crack or lift down the track.
```

**Pattern D - "otherwise it falls apart":**
```
[Service] needs the right materials otherwise it falls apart after a few years. We use products that [benefit] and are [benefit].
```

**Pattern E - "We are licenced":**
```
We are licenced for [service]. [Fragment list]. [Service] needs different [products] to [other service].
```

**Pattern F - "without any dramas":**
```
[Materials list]. These materials need careful handling and the right techniques. We have the experience to work with them without any dramas.
```

**Pattern G - "something a lot of [trade] skip":**
```
[Service] is something a lot of [trade] skip or do badly. We are licenced and make sure there is never any issues with [problem].
```

## Catchy Tagline Template

```
If it [verb], [verb], [verb] or [verb]... we can fix it!
```

Examples:
- Tiling: "If it cracks, chips, leaks or lifts... we can fix it!"
- Plumbing: "If it leaks, drips, breaks or bursts... we can fix it!"
- Electrical: "If it sparks, trips, flickers or fails... we can fix it!"

## Token Patterns for "We" Style

| Pattern | Example |
|---------|---------|
| "dont" without apostrophe | "you dont need to organise" |
| "if you after" (dropped "are") | "if you after a premium look" |
| "when the job done right" (dropped "is") | "No cracking when the job done right" |
| "there is never any issues" (wrong verb) | "make sure there is never any issues" |
| "licenced" British spelling | "We are licenced for pool tiling" |
| "down the track" colloquial | "dont crack or lift down the track" |
| "without any dramas" colloquial | "work with them without any dramas" |
| "falls apart" colloquial | "otherwise it falls apart" |
| No hyphens | "fully stocked vehicles" not "fully-stocked" |
| Missing commas | "professional reliable and affordable" |

# STYLE 2: "I" SOLO STYLE (100% Human)

This style works for tradespersons, consultants, SEO specialists, professional services, and any solo operator or small team.

## Core Sentence Patterns (Verified 100% Human)

### Opening Patterns

**Pattern 1 - Experience Fragment:**
```
[X] years experience doing [service] for [Location] businesses.
```

**Pattern 2 - Familiarity Statement:**
```
I am familiar with every issue there is when it comes to [topic] and know the solution for each.
```

**Pattern 3 - Range Statement:**
```
From [type] in [suburb] to [type] in [suburb] to [type] selling [scope].
```

**Pattern 4 - Belief Conditional:**
```
If you are going to [action] then I believe the most important part is [key point].
```

**Pattern 5 - Contrast Structure:**
```
Instead of using [bad method] which [problem], I [good method].
```

**Pattern 6 - Industry Leaders:**
```
[products/methods] from industry leaders such as [Brand], [Brand] or [Brand].
```

### Body Patterns

**Pattern 7 - This Week Alone:**
```
This week alone, I have [completed/had] [X] [jobs/calls] [description].
```

**Pattern 8 - Average Time:**
```
The average time for us to [action] was [X] minutes/hours/days.
```

**Pattern 9 - All My Quotes:**
```
All my [quotes/reports/campaigns] cover [list], with no hidden [charges/metrics].
```

**Pattern 10 - Average Cost:**
```
The average cost of a [item] is $X, XXX, while a [item] costs around $XXX.
```

**Pattern 11 - Insurance Statement:**
```
I have taken out full insurance and provide warranty for all my work.
```

**Pattern 12 - Dull Details:**
```
The dull details that no one wants to hear but are there when you need them.
```

**Pattern 13 - Punctuality:**
```
I am punctual. I answer my calls. I fix issues.
```

**Pattern 14 - Basic Things:**
```
It may sound like basic things but surprisingly they are not typical in [Location] [industry] industry.
```

**Pattern 15 - Word of Mouth:**
```
Most of my work comes through word, of, mouth. If you do good work, people tell their friends about you. Its as simple as that.
```

**Pattern 16 - Registration:**
```
I am registered with [Company], [Company], and [Company].
```

**Pattern 17 - Since They Know:**
```
Since they know I do the work correctly [result] get approved fast.
```

### Pricing Patterns

**Pattern 18 - If Typical Costs:**
```
If a typical [job] costs $X, XXX, then I would do it for that price.
```

**Pattern 19 - Specific Job Example:**
```
A [size] [job] in [suburb] took [time] and cost $X, XXX in total. The price was for all services [list].
```

**Pattern 20 - Generally Costs:**
```
Generally, [service] cost somewhere between $XXX and $XXX.
```

**Pattern 21 - Time Written Out:**
```
I began at [X] in the morning, and the customer was [benefit] at night.
```

**Pattern 22 - Prices Depend:**
```
[Service] prices depend on the kind of room.
```

**Pattern 23 - Large Scale:**
```
For large scale [service] jobs, I charge $XX, XX per [unit].
```

**Pattern 24 - Minimum Charge Explanation:**
```
For small jobs it is a $XXX minimum charge as I must consider the time needed to travel to the site and the costs of materials for the repair.
```

**Pattern 25 - No Requirement:**
```
There is no requirement for an upfront payment when it comes to [type] jobs.
```

## Token Patterns for "I" Style

| Pattern | Correct | Use This |
|---------|---------|----------|
| Commas in dollar amounts | $2,850 | $2, 850 |
| Commas in ranges | $40-55 | $40, 55 |
| Commas in time ranges | 3-4 months | 3, 4 months |
| "wont" no apostrophe | won't | wont |
| "Its" no apostrophe | It's | Its |
| "dont" no apostrophe | don't | dont |
| "didnt" no apostrophe | didn't | didnt |
| Word of mouth commas | word-of-mouth | word, of, mouth |
| Time written out | 8am | 8 in the morning |
| Simple verbs | resolve | fix |
| Simple adverbs | rapidly | fast |
| Registration not accredited | accredited contractor | registered with |
| Familiar not experienced | I've experienced | I am familiar with |

## Banned Patterns (Both Styles)

### AI Transition Words - NEVER USE
- Furthermore
- Moreover
- Additionally
- However
- Therefore
- Subsequently
- Consequently
- Nevertheless

### AI Marketing Phrases - NEVER USE
- "Transform your space"
- "Your trusted experts"
- "We pride ourselves"
- "We are committed to excellence"
- "State-of-the-art"
- "Cutting-edge"
- "Your trusted partner"
- "Dedicated to delivering"
- "Experience the difference"
- "Meticulous attention to detail"
- "Transforming your vision into reality"
- "Tailored solutions"
- "Comprehensive services"
- "Seamless experience"

### Template Openings - NEVER USE
- "Looking for X? Look no further"
- "Welcome to X"
- "At X, we believe"
- "Are you looking for reliable"

### Quirky Endings - AVOID
- "The principle of fairness prevails"
- "thus claims are approved rapidly"
- Any overly formal or unusual closings
